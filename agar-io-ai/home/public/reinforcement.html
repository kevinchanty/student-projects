Introduction to Reinforcement Learning
Reinforcement learning for AI consist of 2 main parts:
Agent and Environment
Environment
Environment is the place where the AI operate in
In this case, the Agar IO game
For our project, we use OpenAI Gym as the library to create our custom environment of Agar IO
Agent
Agent is the AI itself
we used Stable Baselines 3 to create our own training agent for this project
